# Facebook Messenger App

Pack https://www.messenger.com/ as an App with electron.

Almost all code is generated by Google Gemini-CLI.

## Test

```
npm install
npm start
```

## Build Installer

### Windows

```
npm install
npm run build
```

Installer file: `dist\Facebook Messenger.exe`

### Mac OS

```
npm install
npm run build -- --mac --dir
```

Installer file: `dist/mac-arm64/Facebook Messenger.app`
